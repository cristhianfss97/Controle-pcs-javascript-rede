# Controle-pcs-javascript-rede
Pensando na rotina de empresas que lidam com manutenção, suporte técnico e gestão de máquinas, desenvolvi um sistema focado em organização, praticidade e visual profissional.

Principais funcionalidades:
✔ Cadastro de clínicas e equipamentos
✔ Abertura e gerenciamento de ordens de serviço
✔ Controle de status (pendente, em andamento, finalizado)
✔ Priorização de atendimentos
✔ Exclusão e organização de registros
✔ Geração de histórico e relatórios
✔ Sistema acessível em rede local (várias máquinas)
✔ Interface moderna, leve e intuitiva

Diferencial:
Além da funcionalidade, foquei muito na experiência do usuário — criando uma interface mais humanizada, agradável e com aparência de sistema profissional.

Tecnologias utilizadas:

Node.js
Express
MySQL
HTML, CSS e EJS

Esse projeto foi pensado tanto para uso interno quanto para possível comercialização como sistema.
